# HTTP vs HTTPS

## 📌 Overview
A simple guide explaining the difference between HTTP and HTTPS.

## 🔍 Topics
- Data encryption
- Port numbers (80 vs 443)
- Security risks
- SSL/TLS certificates
- Browser padlock & SEO impact

## ⚖️ Comparison

| Feature | HTTP | HTTPS |
|---|---|---|
| Encryption | None | SSL/TLS encrypted |
| Port | 80 | 443 |
| Security | Vulnerable | Secure |
| Certificate | Not needed | Required |
| Browser Icon | None | 🔒 Padlock |

## 🎯 Conclusion
HTTPS = HTTP + Security. Always prefer HTTPS for safe browsing.
